# Urban Company Clone

A React + Vite clone of the Urban Company landing page and service booking experience.

## Features

- Responsive React UI built with Vite
- Client-side routing with React Router
- Service listing and booking interface
- Tailwind-driven styling and utility classes
- Cart and checkout workflows
- Modular component structure

## Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/akanshayadav077/Urban-company-clone.git
   cd Urban-company-clone
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open the application in your browser at:
   ```text
   http://localhost:5173
   ```
